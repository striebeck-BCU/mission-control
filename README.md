# Mission Control - Custom Tools Task Board

A dynamic task board for building and managing custom tools.

## Features
- Drag-and-drop task management
- Custom tool creation with metadata
- Status tracking: Backlog, In Progress, Done
- Priority levels with visual indicators
- Category/tag system for tools

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Local state management (no backend required)
